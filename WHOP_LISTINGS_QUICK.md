1. Create Whop account at whop.com (free)
2. Create a "company" (your brand)
3. For each product: Create > Digital Product > paste listing copy from PRODUCTS/listings/
4. Set prices to match Gumroad
5. Optionally add a community tier ($49-99/month access to Discord + updates)
