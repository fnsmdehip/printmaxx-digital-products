## Prompt 1: Content Repurposer

Turn one blog post into 10 social posts, 3 email snippets, and a thread.

```
I have a blog post about [TOPIC]. Here's the full text:

[PASTE BLOG POST]

Create the following from this content:
1. 10 standalone social media posts (mix of Twitter, LinkedIn, Instagram captions). Each post should be self-contained and work without context from the original article. Max 280 characters for Twitter versions.
2. 3 email newsletter snippets (2-3 sentences each) that tease the blog post with a "read more" CTA.
3. A Twitter thread (5-7 tweets) that tells the main story from the blog post. First tweet should be a hook that stands alone.

Rules:
- No hashtags unless I ask for them
- No emojis in professional posts
- Each piece should deliver value on its own, not just promote the article
- Use specific numbers from the article when possible
- First sentence of every piece should be the most interesting part
```

**Time saved:** 45-60 minutes per blog post

## Prompt 2: Professional Email Drafter

Generate polished emails from 2-sentence briefs.

```
Write a professional email based on this brief:

Context: [BRIEF - 1-2 SENTENCES ABOUT WHAT YOU NEED]
Recipient: [WHO YOU'RE EMAILING AND YOUR RELATIONSHIP]
Tone: [casual/professional/formal]
Goal: [WHAT YOU WANT THEM TO DO]

Rules:
- Under 150 words
- No "I hope this email finds you well" or similar filler
- First sentence states why you're writing
- Last sentence is a clear, specific ask
- One question maximum
- No bullet points unless listing 3+ items
```

**Time saved:** 15-20 minutes per email (x5/day = 75-100 minutes saved)

## Prompt 3: Research Synthesizer

Paste 3-5 articles and get a structured analysis.

```
Analyze these articles and provide a structured synthesis:

[PASTE ARTICLE 1 TEXT OR URL SUMMARY]
[PASTE ARTICLE 2 TEXT OR URL SUMMARY]
[PASTE ARTICLE 3 TEXT OR URL SUMMARY]

Provide:
1. KEY FINDINGS (3-5 bullet points of the most important takeaways across all sources)
2. CONTRADICTIONS (where the sources disagree and which seems more credible, with reasoning)
3. GAPS (what none of the articles address that seems important)
4. ACTION ITEMS (3 specific things I should do based on this research)
5. ONE-PARAGRAPH SUMMARY (the entire synthesis in 4-5 sentences, suitable for sharing with a colleague)

Rules:
- Flag any claims that seem unsubstantiated
- Note sample sizes when studies are mentioned
- Distinguish between correlation and causation claims
- If a source seems biased, note it
```

**Time saved:** 30-45 minutes per research session

## Prompt 4: Code Reviewer

Paste any code and get bugs, performance issues, and security flags.

```
Review this code. I need:

1. BUGS: Any logic errors, edge cases, or crashes waiting to happen
2. SECURITY: Any vulnerabilities (SQL injection, XSS, exposed secrets, improper auth)
3. PERFORMANCE: Anything that will slow down as data grows
4. READABILITY: Confusing variable names, missing comments on complex logic, inconsistent patterns
5. SUGGESTED FIX: For each issue found, show the corrected code

Code:
```[LANGUAGE]
[PASTE CODE HERE]
```

Context: [WHAT THIS CODE DOES IN ONE SENTENCE]

Rules:
- Rank issues by severity (CRITICAL / MEDIUM / LOW)
- Only flag real issues, not style preferences
- Show the fix, don't just describe it
- If the code is fine, say so. Don't invent problems.
```

**Time saved:** 20-30 minutes per code review

## Prompt 5: Social Post Generator

Describe your topic, get 10 platform-optimized posts.

```
Create 10 social media posts about: [YOUR TOPIC IN ONE SENTENCE]

My niche: [YOUR NICHE]
My audience: [WHO FOLLOWS YOU]

Create:
- 4 Twitter/X posts (under 280 characters, hook-first, no hashtags)
- 3 LinkedIn posts (professional tone, 3-5 sentences, end with a question to drive comments)
- 3 Instagram captions (casual tone, 2-3 sentences, include 1 CTA)

Rules:
- First sentence of every post must be the most interesting part
- Include at least one specific number or data point per post
- No generic advice ("be consistent," "provide value")
- No emojis unless the platform norm requires it (Instagram only)
- Each post should work as a standalone piece
- Vary the format: some lists, some stories, some questions, some bold claims
```

**Time saved:** 30-40 minutes per content batch
